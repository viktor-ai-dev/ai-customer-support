# ai-customer-support

## Core Features

### Data Ingestion & Chunking
- Processes business data such as product catalogs, policies, and FAQs  
- Splits large documents into manageable chunks for efficient AI retrieval  

### Hybrid Search
- Combines semantic embeddings with keyword matching  
- Delivers accurate, context‑aware responses  

### LLM Orchestration
- Prompt engineering to guide LLM responses  
- AI routing to classify queries by type  
- Query rewriting to clarify vague questions  
- Conversation memory for natural interactions  

## System Flow & Capabilities

**End‑to‑end flow:**  
`Question → Query Rewriting → AI Routing → Hybrid Search → LLM Response → Memory`

**Key capabilities:**  
- Multi‑document understanding  
- Context‑aware conversations  
- Scalable AI solutions for e‑commerce
